# 2024 Deep Learning Dataset
- 1160개의 음성 데이터를 직접 취득
- 2 channel audio, 반향음 없음, 야외에서 녹음
- 0도 ~ 180도 사이에서 20도 각도로 class를 구분함 (10개 class)
- 각 데이터는 2초로 구성
- sampling rate 48kHz
